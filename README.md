# node-notes-app

Nodejs app, to add, view, delete the notes

## Simple NodeCLI notes app

### Add note:

`$ node app.js add --title="task1" --body="task details"`

### Remove note:

`$ node app.js remove --title="task1"`

### List note:

`$ node app.js list

### Read note:

`$ node app.js read --title="task1"`
